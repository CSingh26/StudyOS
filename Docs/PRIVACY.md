# Privacy

StudyOS is local-first. Data is stored on-device by default and never sent to a custom backend.

## What is stored
- Courses, assignments, schedules, and notes.
- Optional files selected by the user.
- Optional OCR text for scanned notes.

## Optional services
- Canvas sync (user-authenticated OAuth).
- iCloud sync and sharing (CloudKit).
- Calendar access (EventKit) when enabled.

## Permissions
- Notifications: reminders and timers.
- Calendar: conflict detection and export.
- Files: document picker access on demand.
- Location: optional leave-now alerts.
