# Privacy

StudyOS is local-first. Your data remains on-device unless you explicitly enable cloud sync or sharing.

## Stored Data
- Courses, assignments, calendar events, grades, focus sessions.
- Notes, OCR text, and file references.
- Optional Canvas OAuth tokens (stored in Keychain).

## Optional Services
- **Canvas**: OAuth + API sync.
- **iCloud**: Optional CloudKit sharing for collaboration.
- **Calendar**: EventKit access for conflict detection and export.
- **Location**: Only for optional “leave now” alerts.

## Permissions
- **Notifications**: Reminders for deadlines and study blocks.
- **Calendar**: Conflict detection and export.
- **Files**: Document picker access on demand.
- **Camera**: Document scanning for OCR.
- **Face ID**: App lock.

## Data Export
- JSON/CSV export for assignments and focus sessions.
- User-controlled sharing and deletion.
