# StudyOS

StudyOS is an iOS-first study organizer focused on offline-first workflows, privacy, and accessibility.

## Canvas OAuth setup (preview)
1. Create a Canvas Developer Key in your Canvas instance.
2. Use an OAuth redirect URI such as `studyos://oauth/callback`.
3. Add the Client ID, Redirect URI, Base URL, and scopes in StudyOS Settings.
4. Tap "Connect Canvas" to sign in.
